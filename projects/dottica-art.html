What it does

Dottica.art is a thesis web application that converts uploaded images and videos into pixel art. Instead of a simple downscale-and-blur filter, it uses machine learning-based color quantization and segmentation to decide how each region of the source image should be simplified into pixel blocks, aiming for results that preserve recognizable shapes and color structure rather than just shrinking the resolution.

ML algorithms used

  • SLIC (Simple Linear Iterative Clustering) for superpixel segmentation, grouping neighboring pixels into perceptually meaningful regions before further processing.
  • K-means clustering for color quantization, reducing an image's color palette down to a fixed number of representative colors.
  • Fuzzy C-Means as a soft-clustering alternative to K-means, allowing pixels to belong to multiple color clusters with varying degrees of membership for smoother color transitions.

What I built

My contribution was the backend and the image/video processing workflow: the Python service that receives an uploaded file, runs it through the chosen algorithm (SLIC, K-means, or Fuzzy C-Means), and returns the resulting pixel art. This covered the processing pipeline itself, handling both still images and video frame-by-frame, and the API layer connecting that pipeline to the web frontend.
