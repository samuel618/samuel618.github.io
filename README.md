# Samuel Rodriguez | Data Analyst Portfolio

Static portfolio site showcasing data analysis projects (Python, SQL, Power BI). Plain HTML/CSS/JS, deployed on GitHub Pages.

## Folder structure

```
index.html          Homepage
css/style.css        All site styles
js/main.js           All site scripts
projects/            One HTML page per project (detail pages), e.g. projects/epl-analysis.html
assets/images/       Screenshots, icons, profile photo
```

## Adding a new project

1. Add a new HTML file to `projects/` (e.g. `projects/new-project.html`).
2. Add its screenshots/images to `assets/images/`.
3. Link to it from the projects section on `index.html`.

This keeps each project self-contained so new projects can be dropped in without restructuring the rest of the site.
